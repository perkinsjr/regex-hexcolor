# Regex hexColor ?

Regular expression for matching a hexcolor

## Install

    $ npm install --save regex-hexcolor

## Usage

    const regexHex = require(regex-hexcolor);
    const fullHex = "#FFFFFF";
    const hex = "#FFF";

    regexHex.test(fullHex);
    regexHex.test(hex);
